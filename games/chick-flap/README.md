# Chick Flap

TypeScript + Phaser で実装した、ひよこ主人公の Flappy Bird ライクゲームです。

## 特徴

- 1 ボタンで羽ばたくシンプル操作
- 48x48 ひよこスプライト 3 フレーム + 速度連動回転
- Flappy Bird を意識した速度感とシビアめの当たり判定

## 必要環境

- Node.js 22 以上

## セットアップ

```bash
npm install
```

## ローカル実行

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

## 操作

- 羽ばたく: `Space`
- 羽ばたく: `↑`
- 羽ばたく: 左クリック / タップ
- リスタート: ゲームオーバー後に再入力
